Privacy Policy

Your privacy is paramount to us. Because SiteToApp operates as a clean code compilation tool, we do not require user accounts, track user activity, or store your site credentials.

No Telemetry or Tracking

We do not collect telemetry, usage metrics, or request history from our compiler. When you input a website URL to package into an application, that URL is only used dynamically to generate the template Java config files.

Zero Data Injection

The codebases compiled by our generator contain only native Android/WebView API instructions. We do not inject background trackers, third-party analytics libraries, advertising cookies, or telemetry frameworks. The compiled APK runs independently and connects only to the URL you specified.

Secure Client Data

Standard cookies, local session storage, and database caches handled by your web application are securely isolated inside the client WebView. Neither SiteToApp nor any intermediate script can access this local container space.

Local Control

All source project folders are zipped and downloaded directly to your computer. You retain full control and ownership of the generated code and files.
