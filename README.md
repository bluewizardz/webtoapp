# Site2App Builder

A static website that turns a website URL into a downloadable GitHub Actions project. The generated project can build:

- Android APKs with a native WebView wrapper
- Linux AppImage and deb packages with Electron
- macOS DMG and zip packages with Electron
- Optional Windows EXE artifacts with Electron

Open `index.html` in a browser, enter the URL and app details, then download the ZIP. No backend server is required.
